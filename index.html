ARfrica ARKit Fashion

Sogallo is an artist based in Nairobi. He deals with a lot of line work and paint mediums. For this piece, the idea was to visualize his line art in an Augmented Reality space. App is made in Unity using ARKit plugin, now ARFoundation.

VIRTUAL REALITY

Nairobi Design Week 2022

We had chance to work with Nairobi Design Week during their 2022 event, the assignment was brief, make a VR experience that embodies all the various artists showcasing at the even and add some immersive magic into it! App was created in Unity, Shaders created in Amplify Shader Editor, coded using C#

VRfrica Virtual Reality Fashion

Visualizing African print in a Virtual Reality. The clothing design was done in Marvelous Designer, cloth textures were created procedurally in Substance Designer, assembled together in Unreal Engine 4, and added interaction using Blueprints. Some custom tools were created in Python and Blutilities to speed up the workflow as well.

Hand Tracking HTC Vive Leap Motion - Anatomy

Before the Meta Quest 2 or even 3 and hand tracking were a thing, the Leap motion was what we used to get hand tracking in VR. Made un Unreal Engine 4, interaction using Blueprints but some C++ for storing all the data for the various bones.

Archery VR! ChemLab VR For HTC Vive.

ChemLab VR is a game where students get to play games based on the periodic table, in this particular version, there is a bow and arrow as well a cross-bow, the idea is to shoot at the correct answer to score the highest points. Made in Unreal Engine 4 with interaction done in Blueprints, the questions are created in C++ and read in to Blueprints.

METAVERSES

AltSpace VR - Club Metaverse

This was a club that I got commisioned to optimize for Altspace VR, originally it was a high poly asset made for PC, I also did the texturing, lighting and created the custom shaders using Unity Shader Graph

AltSpace  VR - Rangi

Rangi means Color in Swahili, an East African language, the idea behind this was to create a gallery that reveals the color of the artwork when close to it, hence the name "Rangi". I created the shaders in Unity Shader Graph, Light baking was done in Blender.

Altspace VR - Floating Mountains  

The main test with this was using UDIMs to create a very large landscape that would have different ground type, this was done using Substance Painter and Blender for the light baking.
